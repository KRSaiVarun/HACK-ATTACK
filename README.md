## 📌 Overview

**HACK-ATTACK** is a modern hackathon/event website designed to showcase a technology-driven competition. It highlights domains like AI/ML, Web Development, Cybersecurity, IoT, and Blockchain while providing participants with event details, schedules, and resources.

The platform is built to create an engaging experience for users interested in innovation, collaboration, and problem-solving through hackathons.

Hackathons like this aim to bring developers together to build solutions in areas such as healthcare, fintech, smart cities, and more ([Hack Attack][1])

---

## ✨ Features

* 🌐 Interactive and responsive UI
* 📅 Event schedule and timeline
* 🧠 Multiple domains (AI/ML, Web, Cybersecurity, IoT, Blockchain)
* 📂 Problem statements download section
* 🏆 Hackathon details, benefits & guidelines
* 🤝 Sponsor and partner showcase
* ❓ FAQ section for participants
